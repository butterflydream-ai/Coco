Coco / 機能 / 為替

複数の通貨を、一行で合算

`$100 + ¥800 + €20` をライブレートで合計し、見ている通貨ごとに——推移つきで表示。

⬇ macOS 版をダウンロード

一行に、いくつもの通貨

たいていの換算ツールは、一つの金額・一つの通貨だけ。Coco は式まるごとを受け取ります：`$100 + ¥800 + €20` を一つずつライブレートで換算して合計——本物の合計で、三回ばらばらに調べたものではありません。

算術なので `−`、`×`、`÷` も使えます：`$100 × 3`、`(¥5000 + €10) ÷ 2`。記号とコードは自由に混ぜられます。

見ている通貨すべて、推移つきで

合計はデフォルトの一覧に一度に並びます——CNY、USD、EUR、JPY、KRW ほか——各行に小さなスパークラインと前回からの変化がついて、数字だけでなく方向が見えます。

その一覧は設定で選んで並べ替え、あるいはインラインで対象を一つ指定：`$100 to EUR`。

書いたとおりに打てばいい

記号（`$`、`¥`、`€`、`₩`）、ISO コード（`USD`、`JPY`）、短い別名（`jp`、`kr`、`rmb`）——金額の前でも後ろでも。`100 usd to eur` は `$100 → €` と同じに読まれます。あなたが合わせる必要はありません。

ライブで、しかも正直に

レートはバックグラウンドで静かに更新され、結果に日付が出るので、コピーする数字は最新です。オフラインでは直近の良いレートに戻り、推測ではなくその日付の“時点”と示します。

タブも、換算サイトも、Cookie バナーもなし。ほかで押すのと同じホットキーの先にあります——数字と単位の純粋な計算はこちら →

よくある質問

違う通貨どうしを足せますか？

はい。$100 + ¥800 + €20 のように一つの式を打つと、Coco がそれぞれをライブレートで換算して合計します。−、×、÷ にも対応し、$100 × 3 や ¥5000 ÷ 2 も使えます。

結果はどの通貨で表示されますか？

設定したデフォルトの一覧——設定で並べ替え可能——か、$100 to EUR のように対象を一つ指定します。各行に小さな推移と前回からの変化が出ます。

レートは更新されますか？

はい。レートはバックグラウンドで更新され、結果に日付が出ます。オフラインでは直近のレートに戻り、その日付の“時点”と示します。

正確な記号が必要ですか？

いいえ。記号（$、¥、€）、ISO コード（USD、JPY）、短い別名（jp、kr、rmb）のどれでも、金額の前でも後ろでも使えます。
